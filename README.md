This repository contains an HR Analytics Dashboard built in Power BI to help organizations monitor, analyze, and understand employee attrition trends.  
The dashboard provides insights into workforce demographics, attrition distribution, job roles, education backgrounds, and salary metrics â€” enabling data-driven decision-making for HR teams.

 Project Overview

Employee attrition is a critical challenge for HR departments. This dashboard consolidates key HR metrics into a single interactive interface, allowing managers to:

- Track total employees and overall attrition  
- Understand attrition rate by age, gender, department, and job role  
- Analyze workforce composition by education  
- Compare average age, salary, and tenure of employees  
- Identify high-risk segments for targeted retention strategies
